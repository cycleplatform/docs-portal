:::note Create Workflow

- Click the Environments tab on the navigation menu to the left.
- Select the **Create Environment** button.
- In the form, enter a name for this environment (usually the name of your application you are building) and an optional description.
- Click Create Environment. You will be taken to the environment dashboard.

:::

![environment create form](https://static.cycle.io/portal-docs/environments/create.png)

* **Name** - Human readable name for the environment. 
* **Cluster Selection** - Select the cluster this environment will use resources from.
* **Access Controls** - Set environment access controls (more info [here](/reference/hubs/access-controls))
* **Advanced** - For users that would like to use IPv4 networking as their primary protocol in an environment, legacy networking is available as an option during environment create.
